8. 성능 점검

8.1. SQL 표준 점검

8.1.3. SQL 표준 준수율 현황/추이

개발자가 작성한 SQL표준 점검 결과에 대한 현황 및 추이에 대해 다음과 같은 그래프를 보여준다.

표준 준수율 추이

품질검토 수행일자별 표준 준수율, 프로그램 본수, 미준수 건수 추이

업무별 현황

최종 수행된 품질검토결과에 대한 프로젝트 전체의 SQL 본수, 미준수 본수, 표준 준수율, SQL 증감, 미준수 증감, 준수율 증감 현황 및 업무별 금회차/전회차 준수율, 프로그램 본수, 미준수 건수 현황

품질지표별 미준수 현황

최종 수행된 품질검토 결과에 대한 프로젝트 전체 / 업무별 품질 지표별 미준수 건수

업무별 표준 준수율 추이

품질검토 수행 일자별 프로젝트 전체 / 업무별 표준 준수율 추이

품질지표별 미준수 추이

품질검토 수행 일자별 프로젝트 전체 / 업무별 품질지표 단위의 미준수 건수 추이

업무별 현황(전체) 마우스 클릭

업무별 그래프를 보기 위해 업무별 현황(전체) 화면에서, 업무별로 챠트 그래프를 클릭하면 품질지표별 미준수 현황, 업무별 표준 준수율 추이, 품질지표별 미준수 추이의 챠트 그래프는 업무별로 반영된다.
