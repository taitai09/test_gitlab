11.1.2. 신규 SQL 타임아웃 예측

SQL에 대하여 타임아웃이 예측되는 시점 정보를 제공한다. 타임아웃 시간은 60초로 설정되었으며 업무 요건에 맞게 DB 단위로 변경이 가능하다.

신규 SQL 타임아웃 예측 조회

DB / 발생일(최초수행일) / 예측 일 등의 조건을 선택하여 검색하면 타임아웃이 예상되는 신규 SQL 정보를 조회한다.

조회된 리스트를 선택하면 타임아웃 예측 시점을 확인할 수 있는 추이 그래프와 SQL상세 분석 정보가 제공된다.

신규 SQL 타임아웃 예측 성능분석

[성능분석] 탭을 선택하면 앞서 그리드에서 선택한 정보(타임아웃이 예측되는 SQL)에 대하여 분석된 SQL FULL Text, Bind값에 따른 성능정보, Plan, Outline, SQL Statistics History 등의 정보를 보여준다.
