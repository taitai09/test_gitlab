13.1. Open-POP 설정

13.1.4. APM 접속 관리

APM 접속 정보를 조회하고 신규로 추가하거나 수정/삭제할 수 있다.

APM 접속 조회

화면의 좌측에서 APM 접속 정보를 조회할 수 있다.

APM 접속 추가

화면 우측 APM 추가/수정 영역에서 [초기화] 버튼을 클릭하고 신규로 추가할 APM 접속 정보를 입력한 후 [저장] 버튼을 누른다.

APM 접속 수정

화면의 좌측에 있는 목록에서 수정할 APM을 선택하면 우측 APM 추가/수정 영역에 선택한 정보가 나타나고 수정할 내용을 변경한 후 [저장] 버튼을 누른다.

APM 접속 삭제

화면의 좌측에 있는 목록에서 삭제할 APM을 선택하면 우측 APM 추가/수정 영역에 선택한 정보가 나타나고 [삭제] 버튼을 누른다.
