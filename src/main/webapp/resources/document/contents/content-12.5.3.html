12.5. SPACE 분석

12.5.3. 파티셔닝 대상 분석

대용량 테이블을 보다 효율적으로 관리하기 위해 테이블을 작은 단위로 나눔으로써 데이터 엑세스 작업의 성능 향상을 유도하고 데이터 관리를 보다 용이하게 하기 위해서 테이블 파티션이 필요하다. Open-POP은 테이블 사이즈와 사이즈 증가 추이를 고려하여 파티션 대상을 식별하고 access path기반의 컬럼 사용 효율성을 분석하여 파티션 키 컬럼을 추천한다.

파티셔닝 대상 분석

파티셔닝 대상을 식별하기 위하여 TOP Size Table 현황, TOP Size Table Histor, 전체 테이블 목록을 보여준다.
파티셔닝 대상분석 화면은 세 개의 카테고리로 구성되었다.

  • - TOP Size Table : 사이즈 TOP 10 테이블의 사이즈 크기 차트
  • - TOP Size Table History : 사이즈 TOP 10 테이블의 사이즈 증가 추이
  • - Table List : 테이블 건수가 0건 이상인 테이블 목록과 최근 6개월의 사이즈

파티셔닝 대상 상세 분석

파티셔닝 대상 상세분석화면은 아래 세 개 카테고리를 보여준다.

  • - Table Size History : 파티션 대상 테이블의 테이블 사이즈 변경 추이
  • - Access Path : 파티션 대상 테이블의 조건절에 대한 Access Path 패턴
  • - Partition Key Recommendation : Access Path 정보를 기반으로 성능 측면에서 권고하는 Partition Key Column
