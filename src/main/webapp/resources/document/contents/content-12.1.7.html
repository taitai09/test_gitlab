12. DB 진단/점검

12.1. 일 예방 점검 – DB 상태 점검

12.1.7. Alert

DB에서 발생하는 심각한 Error 또는 Incident(사건), 스케쥴러 작업 실패 등 관리자가 인지해야 하는 경고성 이벤트를 점검한다.

Alert Log Error

1일 동안 Alert Log에 기록되는 오라클 에러 목록을 보여준다.

Active Incident

1일 동안 Alert로그에 기록되는 Error를 Incident(사건)과 Problem(문제) 로 구분하여 보여준다.

  • Incident : alert log에 기록되는 error(problem)때문에 발생하는 event 하나하나
  • Problem : "incident"의 원인이 되는 근본적인 문제이며, 일반적으로 하나의 problem는 many incidents를 가진다.

Outstanding Alert

1일동안 DB에서 발생한 이슈를 점검한다. 확인할 수 있는 대표적인 이슈는 아래와 같다.

  • 테이블스페이스 Full
  • ora-01555 snap shot too old
  • resumable session 이상발생

DBMS SCHEDULER JOB FAILED

1일 동안 DB 스케쥴러 작업 실패목록을 보여준다.
