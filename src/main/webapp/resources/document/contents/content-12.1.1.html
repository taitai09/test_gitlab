12.1. 일 예방 점검 – DB 상태 점검

12.1.1. Database 점검

Database Stauts

Database 상태를 점검한 결과이며 검색기능을 통해 다른 DB의 상태를 확인할 수 있다.
OPEN_MODE 가 “READ WRITE”가 아닌 목록을 보여준다..

Expired Grace Account

오라클 계정 상태가 EXPIRED(GRACE) 인 계정목록을 보여준다. 계정이 잠기면 로그인이 안되기 때문에 장애상황이 발생할 수 있다. 계정상태와 잠금시간, 남은시간을 함께 보여준다.

Modified Parameter

인스턴스 기동 이후에 변경된 파라미터 목록을 보여준다.

New Create Object

전일에 새로 생성된 오브젝트 목록을 보여준다.
