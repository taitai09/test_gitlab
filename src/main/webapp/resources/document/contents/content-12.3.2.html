12.3. 장애 예측(자원한계점) 분석

12.3.2. CPU 한계점 예측

DB INSTANCE의 CPU 사용률이 한계점에 도달하는 시점 정보를 제공한다. 이 정보를 토대로 DBA는

  • - 자원 증가 추이 분석
  • - 튜닝
  • - 자원 증설

등의 의사 결정을 위한 정보로 활용할 수 있다.

CPU 한계점 예측 조회

예측 기준을 선택하여 검색하면 DB별 INSTANCE 의 CPU 사용률 한계점 도달 시점 정보를 조회한다.

조회된 정보를 선택하면 상세 화면으로 이동한다.

CPU 한계점 예측 상세 조회

상세 화면은 한계 사용률, 사용률, 사용률 추이, 예측 추이 값을 조회하여 그래프로 제공한다. 조건을 재설정하여 검색하면 재 조회가 가능한다.

CPU한계점 예측 차트 범례의 의미는 아래와 같다.

  • - 한계 사용률 : CPU 사용 한계 임계값이며 관리자가 시스템(DB) 단위로 설정
  • - 사용률 : 실제 CPU 사용률(AWR 기반)
  • - 사용률 추이 : 실제 CPU 사용률을 기반으로 분석된 추이(Trend)
  • - 예측 추이 : 실제 CPU 사용률을 기반으로 분석된 미래 사용률 예측 추이(Trend)
