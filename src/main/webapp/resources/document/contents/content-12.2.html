12. DB 진단/점검

12.2. 일 예방 점검 – 일 점검

DB별 DB, INSTANCE, SPACE, OBJECT, STATISTICS, ALERT, LONG RUNNING WORK점검 관점에 대한 일일 점검 결과와 함께 문제 해결을 위한 조치 스크립트를 제공한다.

DB 상태 점검

일일 DB 점검 결과에 대한 요약 정보를 제공한다.

DB 상태 점검 조회

메뉴 클릭 시 자동 조회되며 DB 그룹, 심각도를 선택하여 조회할 수 있다.

‘심각도’의 의미는 아래와 같다

  • (■)긴급조치 : 긴급성 조치 필요 건
  • (■)조치필요 : 비 긴급성이나 조치가 필요한 건
  • (■)확인필요 : 점검결과 확인이 필요한 건
  • (■)정상 : 점검결과 조치할 건이 없음
  • (■)미수행 : 일점검이 수행되지 않음(관리자에게 확인 필요)

DB 상태 점검 상세

점검항목그룹별로 점검항목에 대한 상세 점검결과를 제공한다.

DB 상태 점검 상세 조회

‘DB 상태 점검’ 화면에서 점검그룹 클릭 시 탭이 생성되어 조회된다. 조회된 탭에서 아래의 점검그룹항목 탭 클릭 시 해당 점검항목의 상세한 점검결과를 제공한다.

DB 상태 점검 현황

특정 기간 동안의 DB 상태 점검 결과를 조회하는 화면이다.

DB 상태 점검 현황 조회

‘DB 상태 점검 현황’ 탭 클릭 시 자동 조회된다. 1주일, 1개월을 선택할 경우 해당 기간 내의 정보가 조회되며 특정 기간을 조회하고자 하는 경우 기준일자를 선택하여 조회하면 된다.
