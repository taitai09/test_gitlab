13.3. 기준정보 설정

13.3.3. 업무 기준정보

업무별로 적용되는 기준정보를 확인하고 설정한다. 최초 설정 값은 Open-POP에서 가이드하는 기본값을 사용하며 변경이 필요한 경우 설정 값을 변경한 후 [적용] 버튼을 클릭한다.

기준정보 유형은 아래와 같다.

  • - APM성능진단 : Timeout Error Code, 응답시간증가율, 응답시간증가 비교대상일수, DBIO 추출 정규식
  • - 성능점검 : Application Threadhold Elapsed Time, Application Threadhold Block, Application ThreadHold Compute Method, DBIO Threadhold Elapsed Time, DBIO Threadhold Blocks, DBIO ThreadHold Compute Method
  • - Test Case Library : 최소 Test Case 수
