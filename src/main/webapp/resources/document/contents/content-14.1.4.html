14. 기타

14.1. 기타 기능

14.1.4. ToDo

관리자 / DBA 담당자에게 앞으로 해야 할 일(To do)이 대시보드에 조회된다.

  • - [DB 예방 점검 현황] 탭: 인스턴스별 DB 예방 점검 현황
  • - [성능 리스크 진단 현황] 탭: DB별 플랜변경, 신규 SQL, LITERAL, TEMP과다 사용, FULL SCAN 사용 등 SQL 성능 진단 현황
  • - [튜닝 진행 목록] 탭: DB별 튜닝 진행 목록
  • - [튜닝 지연 목록] 탭: 지연된 튜닝 목록
  • - [긴급 조치 대상] 탭: 일 예방 점검 결과 긴급 조치해야 할 목록
  • - [오브젝트 변경 현황] 탭: DB별 최근 일주일 간 변경한 오브젝트 현황
  • - [튜너미할당요청목록] 탭: 튜닝 담당자를 지정하지 못한 SQL 목록
  • - [튜너별할당현황] 탭: 튜닝 담당자별 할당 현황
