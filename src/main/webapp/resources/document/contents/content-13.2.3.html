13.2. 사용자 관리

13.2.3. 업무 관리

사용자 업무를 관리하는 화면이다. 업무TREE는 현재 설정된 업무 구조를 보여주고 우측의 업무 추가/수정을 통해 업무구조와 정보를 변경할 수 있다. 업무TREE에서 업무를 선택하면 사용자 목록을 화면 하단 “업무별 사용자 내역” 에 보여주며 여기서 해당 업무의 리더를 변경할 수 있다.

업무 리더는 성능개선관리에서 해당 업무내의 모든 개발자에 대한 성능개선현황을 볼 수 있고, 튜닝 건에 대하여 개발 담당자를 변경할 수 있다.

업무 트리 조회

전체 업무의 목록을 트리 형태로 조회한다.

업무 추가

업무 추가/수정 영역에서 [초기화] 버튼을 클릭한 후에 상위업무, 업무코드, 업무명, DB정보를 입력한 후 [저장] 버튼을 클릭한다.

업무 수정

업무TREE에서 업무를 선택한 후에 업무코드, 업무명, DB 정보를 변경한 후 [저장] 버튼을 클릭한다.

업무리더 여부 변경

업무별 사용자내역 영역에서 리더로 지정할 개발자를 선택(체크)한 후 [저장] 버튼을 클릭한다.
