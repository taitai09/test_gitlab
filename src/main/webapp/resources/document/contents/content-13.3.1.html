13. 설정

13.3. 기준정보 설정

Open-POP 운영에 필요한 다양한 기준정보를 설정한다. 설정정보는 세 가지로 구분하여 관리한다.

  • - 시스템 기준정보 : Open-POP 솔루션 운영에 필요한 Global 설정
  • - DB 기준정보 : 관리대상 DB별 설정
  • - 업무 기준정보 : 관리대상 업무별 설정

13.3.1. 시스템 기준정보

SQL진단시간 애플리케이션 진단시간 등 스케줄러 작업시간 설정, 성능개선관리의 alert 지연 일수 등 Open-POP 솔루션 운영에 필요한 Global 설정 값을 확인하고 변경한다.

최초 설정 값은 Open-POP에서 가이드하는 기본값을 사용하며 변경이 필요한 경우 설정 값을 변경한 후 [적용] 버튼을 클릭한다.

기준정보 유형은 아래와 같다.

  • - 스케줄러 : DBIO-SQL 매핑시간, SQL OUTLINE 추출시간, DB파라미터변경적재시간, SQL성능분석시간 등
  • - 성능개선관리 : 튜닝지연 ALERT 일수, 튜닝반영지연 ALERT 일수
