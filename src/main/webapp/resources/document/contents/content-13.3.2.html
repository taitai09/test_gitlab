13.3. 기준정보 설정

13.3.2. DB 기준정보

관리대상 DB별로 적용되는 기준정보를 확인하고 설정한다. 최초 설정 값은 Open-POP에서 가이드하는 기본값을 사용하며 변경이 필요한 경우 설정 값을 변경한 후 [적용] 버튼을 클릭한다.

기준정보 유형은 아래와 같다.

  • - 인덱스 자동설계 : NDV Ratio, Null처리 기준건수, Blocks, NUM_ROWS
  • - 셀프튜닝 : 셀프테스트 타임아웃 시간, 셀프테스트 Threshold Elapsed Time, 셀프테스트 Threshold Blocks
  • - 스케줄러 : DB예방점검 시간, SPACE진단 수행시간
  • - SQL성능진단 : Literal 기준건수, TEMP사용 임계값, 성능저하 비율, 성능 비교일수 등
  • - SPACE 진단 : 파티션대상 테이블사이즈, 파티션대상 테이블건수, 파티션대상 월 증가사이즈, 파티션대상 월 증가건수, 파티션대상 Read Activity 등
  • - 장애예측관리 : 장애예측 비교일수, CPU자원한계임계값, SEQUENCE사용한계임계값, TABLESPACE사용한계임계값
