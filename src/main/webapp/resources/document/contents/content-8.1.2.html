8.1.2. SQL 표준 품질점검 설정

[SQL 품질점검 지표 관리] 탭

SQL 표준 품질점검 지표 정보를 조회 및 추가하거나 수정한다.

엑셀

현재 데이터를 엑셀형식으로 다운로드 하여 보여준다.

데이터 등록

[초기화] 버튼을 누른 후 데이터를 순서대로 입력 한다.

데이터 수정

원하는 데이터를 클릭 후 수정한다.
수정 시 음영영역은 데이터수정이 불가하다.

데이터 삭제

원하는 데이터를 클릭 후 삭제한다.

[SQL 품질점검 RULE 관리] 탭

SQL표준에 대한 품질점검은 고객사 마다 점검대상 및 점검RULE이 상이할 수 있다.
유지보수 편의성을 위해 변경이 예상되는 RULE을 Table에서 관리하고, 메뉴에서 해당 SQL을 호출하는 방식으로 프로그래밍 되어 있다.
본 가이드에 있는 표준 RULE을 셋팅하고 작성 규칙을 준수하여 RULE을 수정해야 한다. RULE의 수정은 Openmade가 수행한다.

[프로젝트 SQL 품질점검 지표 관리] 탭

프로젝트 별로 SQL품질 점검 지표 코드를 조회 및 추가하거나 수정할 수 있다.

검색조건

검색조건으로는 프로젝트와 적용여부가 있다.

프로젝트 조회

프로젝트 SQL 품질점검 지표 조회

프로젝트에 해당하는 SQL 품질점검 지표를 조회한다.

프로젝트 표준 지표 적용

선택된 프로젝트의 품질점검RULE에 대하여 표준 RULE을 일괄 적용한다.

엑셀

조회된 데이터를 엑셀로 다운로드한다.

프로젝트 SQL 품질점검 지표 조회

검색된 데이터를 그리드에서 클릭하였을 때 아랫부분에 상세 정보를 보여준다.

프로젝트 SQL 품질점검 지표 수정

상세 정보 조회 모드에서 적용여부를 적용, 비적용으로 변경할 수 있다.

초기화

상세 정보 조회 항목들을 초기화한다.

프로젝트 표준 RULE적용
