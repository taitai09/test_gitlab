11. 성능 진단

11.3. DB 종합 진단

주간 및 월간 단위로 DB 점검, 오브젝트 진단, 파라미터 진단, SQL 성능 진단, 장애 예측 분석 결과에 대한 점검 결과 요약 및 조치 대상에 대한 상세한 DB 종합 진단 결과를 보고서로 제공한다.

DB 종합 진단 조회

DB, 기준 일자, 주간 진단 또는 월간 진단을 선택하여 DB 종합 진단 보고서를 조회한다.

‘결과 평가’ 항목의 의미는 아래와 같다

  • 우수( ) : 조치 사항 없음
  • 양호( ) : 금일 조치 사항이 없지만 1개월 동안 조치 사항 있음
  • 조치필요( ) : 비 긴급성 조치 필요
  • 긴급조치( ) : 긴급성 조치 필요
  • 확인필요( ) : 점검결과 확인 필요

DB 상세 진단 결과 확인

‘점검 항목‘ 명을 클릭하면 상세 점검 결과 페이지도 이동한다.

‘결과 평가’ 항목이 (비 긴급성 조치 필요), (긴급성 조치 필요)인 항목에 대해서 우선 확인 및 조치를 진행하고, 진단 결과 (확인필요) 항목에 대해서도 확인이 필요하다.
