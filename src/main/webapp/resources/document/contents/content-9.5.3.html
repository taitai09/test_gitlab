9.5.3. 테이블/인덱스 사용 SQL 분석

테이블 사용 SQL 분석 탭

LOCK, I/O 경합 등의 성능적인 이슈가 발생하여 특정 테이블을 참조하는 SQL에 대한 상세 분석이 필요한 경우, 테이블 사용 SQL 분석을 통해 성능 이슈의 원인을 발견하고 이슈를 해결하는데 유용하게 활용할 수 있다.
