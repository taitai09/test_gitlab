11.1. SQL 성능 진단

11.1.4. TOP SQL 추이/현황

검색영역에서 조건값을 입력하여 검색하면 좌측영역에 다음과 같은 6가지의 그래프를 보여준다.
[CPU], [Elapsed Time SQL], [CPU Time SQL], [Executions], [Logical Reads], [Physical Reads]

[CPU] 차트를 제외한 각 차트의 그래프를 클릭하면 우측에 Module, Action, Parsing Schema 각각에 대한 추이를 볼 수있으며, 각 탭의 Module, Action, Parsing Schema를 클릭시 ‘AND’ 조건으로 TOP SQL이 하단에 출력된다.

차트별 현황 마우스 클릭

좌측 현황 화면에서 각 추이차트 그래프를 클릭하면 각 차트별, 날짜별로 우측영역에Module, Action, Parsing Schema에 대한 현황을 보여준다.

Module, Action, Parsing Schema 그리드 클릭

TAG영역에 Module, Action, Parsing Schema 에 대한 정보가 각1개씩 TAG 영역에 추가되며, 추가/삭제 시 마다 ‘AND’ 값 조건으로 TOP SQL 이 조회된다.

엑셀 다운로드

엑셀 다운로드시 최종 검색 조건이 반영된 TOP SQL이 반영된다.
