14.1.11. 점검 그룹 DB 관리

모니터링 그룹에서 관리되는 DB를 조회하고 신규로 추가하거나 수정/삭제할 수 있다.

점검 그룹 DB 관리 조회

화면의 상단에서 조회 조건으로 목록을 조회할 수 있다.

점검 그룹 DB 관리 신규 추가

화면 하단에서 [초기화] 버튼을 클릭하고 사용할 그룹명, DB명을 선택하고 정렬순서 입력 후 [저장] 버튼을 클릭한다.

  • 그룹명과 DB 명 중복 시 등록 불가
  • 동일 그룹 아이디에 정렬순서 중복 시 등록 불가

점검 그룹 DB 관리 수정

화면의 상단에 있는 목록에서 수정할 작업을 선택하면 하단에 선택한 정보가 나타나고 수정할 내용을 변경한 후 [저장] 버튼을 누른다.

  • 동일 그룹 아이디에 정렬순서 중복 시 수정 불가

점검 그룹 DB 관리 삭제

화면의 상단에 있는 목록에서 삭제할 작업을 선택하면 하단에 선택한 정보가 나타나고 [삭제] 버튼을 클릭한다.
