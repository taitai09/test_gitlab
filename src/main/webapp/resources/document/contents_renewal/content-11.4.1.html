11.4. SQL 성능 분석

11.4.1. 모듈 성능 분석

일반적으로 배치 프로그램은 DBMS_APPLICATION_INFO.SET_MODULE 오라클 패키지를 통해 프로그램을 DB에서 식별할 수 있도록 모듈 명을 지정한다. 간혹 온라인 프로그램도 모듈 명을 지정한다.

모듈 성능분석은 모듈에서 수행되는 모든 SQL에 대한 성능을 분석하는 화면으로 장시간 수행하는 배치 프로그램에서 부하가 큰 SQL을 식별하는데 효율적으로 사용한다.

모듈에서 수행한 전체 SQL목록과 함께 TOP SQL현황과 단위 SQL의 수행이력을 비교 분석하고, SQL성능분석화면과 연계하여 상세분석이 가능하다.

모듈성능분석 조회

DB와 모듈명을 입력한 후 검색하면 입력 값을 Like검색하여 모듈에서 수행하는 모든 SQL을 조회한다. [SQL List] 탭에서 SQL을 선택하면 하단에 SQL 성능이력이 보여지고, SQL성능이력을 선택하면 새 탭에서 SQL성능분석화면으로 이동한다.([SQL성능 분석] 화면 설명 참조)

TOP SQL Stat 조회

[TOP SQL Stat] 탭을 선택하면 모듈내에서 TOP 10 SQL에 대한 성능정보를 차트로 보여준다.
TOP SQL Stat 항목은 아래와 같다.

  • - Total Elapsed Time
  • - Avg Elapsed Time
  • - Avg Buffer Gets
  • - Avg Disk Reads
  • - Executions
