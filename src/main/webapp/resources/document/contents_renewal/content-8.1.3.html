8. 품질 점검

8.1. SQL 표준 점검

8.1.3. SQL 표준 점검 설정

개요

SQL 표준 점검에 대한 지표 설정 및 점검 RULE을 관리하는 기능을 제공한다. 프로젝트별 점검지표 선택 및 필요한 경우 RULE을 변경하여 적용할 수 있다.

NOTE

SQL 표준 점검은 등록된 지표 및 RULE을 참조하여 실행한다. 따라서 품질점검 지표의 등록 및 RULE의 변경은 Openmade가 수행한다. 고객사에서 임의로 수정할 경우 오류가 발생할 수 있으므로 주의가 필요하다. 단 불필요한 품질지표의 삭제 및 품질점검지표명, 품질 점검 내용, 해결 방안 내용에 대한 수정은 고객사에서 수정이 가능하다.

고객사의 RULE 변경 필요 시 Openmade가 방문하여 직접 변경할 수 없을 경우, 변경된 암호화된 RULE을 제공한다. RULE 반영 시 암호화된 RULE의 처음과 마지막에 공백 등 제공된 RULE이외의 문자가 포함되면 오류가 발생하므로 주의가 필요하다.
