9. 성능 검증

9.1. 배포전 사용자 성능 검증

9.1.4. 성능 검증 예외 처리 이력

[성능 점검 예외 요청], [성능 점검 예외 삭제] 예외처리/삭제했던 이력이 보여지는 화면이다.

[프로그램 소스] 탭

  • - 업무, 배포요청일자를 입력해야만 데이터 조회가 가능하다.
  • - 배포요청일자를 체크해지한다면 검색조건(SQL식별자, 프로그램명, 거래, 프로그램설명)을 입력해야만 데이터 조회가 가능하다.

[성능 점검 결과] 탭

성능점검결과가 보여진다.

[예외 요청] 탭

그리드에 있는 데이터를 클릭하면 예외요청사유, 처리방법, 요청자, 상세사유 등이 보여진다.

[예외 처리] 탭

예외처리/삭제 건 등에 대해서 어떻게 처리하였는지 상세하게 볼 수 있다.

엑셀 다운로드

검색조건이랑 동일한 데이터가 엑셀로 출력된다.
