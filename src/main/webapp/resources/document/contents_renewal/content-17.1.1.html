17. Eclipse Plug-in

Open-POP 기능 중 Eclipse에 Plug-in 된 기능으로 고객사 및 Eclipse 버전에 따라 아래 설명된 매뉴얼이 상이할 수 있으므로 고객사에 배포된 Eclipse Plug-in 가이드를 참조한다.

17.1. SQL 표준 점검

개발자 본인이 작성한 SQL에 대한 표준 준수 여부를 점검한다.

17.1.1 Preferences 설정

SQL 표준 점검을 위한 정보를 설정한다.

Preferences 설정

상단 메뉴에서 Window > Preferences를 클릭한다.
Preferences 팝업창에서 Open-POP을 클릭한 후 가이드 된 설정 정보를 입력하고 [Apply] 또는 [Apply and Close] 버튼을 클릭한다.

  • Open-POP Server IP: Open-POP Server의 IP
  • Open-POP Server Port: Open-POP Server의 Port
  • Project ID: 해당 프로젝트에 부여된 ID
  • Workjob Code: 해당 프로그램의 업무코드
  • DB Name: 프로그램이 실행될Database명
  • DB User: 프로그램이 실행될Database 접속자명

설정 정보가 입력되지 않았을 경우 아래와 같은 메시지 박스가 뜬다.
