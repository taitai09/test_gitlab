15. 기타

15.2. FAQ

Q. 비밀번호를 잊어버렸어요?

A. 관리자에게 문의하여 비밀번호 초기화를 요청해 주세요. 관리자 비밀번호 초기화 후 관리자가 알려준 임시비밀번호를 입력하고 로그인합니다. 로그인 시 비밀번호를 변경하는 화면이 뜨고 비밀번호를 변경하시면 됩니다.

Q. 사용자 등록은 어떻게 하나요?

A. Open-POP 접속 후 로그인 화면 > [신규 사용자 등록] 버튼 클릭 > 사용자 정보 등록 후 [등록] 버튼을 클릭합니다. 등록된 사용자는 관리자 승인 후 이용이 가능합니다.

Q. 신규 프로젝트에서 SQL 품질 점검을 사용하려면 어떻게 해야 하나요?

A. 신규 프로젝트 등록 > 프로젝트별 SQL 품질지표 선정 > 프로젝트별 SQL 품질점검 RULE 등록 순으로 진행합니다.

  • ① 신규 프로젝트 등록: 설정 > 기준정보 설정 > 프로젝트 관리 > [프로젝트 관리] 탭 메뉴 이용(이미 프로젝트가 등록되었을 경우 생략)
  • ② 성능 점검 > SQL 표준 점검 > SQL 표준 품질점검 설정 > 프로젝트 SQL 품질점검 지표 관리 탭 메뉴 이용
  • ③ 성능 점검 > SQL 표준 점검 > SQL 표준 품질점검 설정 > 프로젝트 SQL 품질점검 RULE 관리 탭 메뉴 이용

Q. SQL 튜닝 대상을 자동으로 선정할 수 없나요?

A. 자동 선정 메뉴를 통해 지속적이고 정기적인 튜닝이 필요한 경우 DB 단위로 자동선정주기, 선정범위, 임계값 등의 조건을 설정하여 TOP SQL을 선정하고 스케쥴링 할 수 있습니다. 물론 수동 선정도 가능합니다.(메뉴: 성능 점검 > SQL 성능 점검 > 자동 선정)

Q. 개발자 스스로 SQL의 성능을 점검할 수 있나요?

A. 개발자가 작성한 SQL의 성능 품질을 일정 수준 확보하기 위하여 개발자 스스로 튜닝할 수 있는 환경을 제공합니다. SQL 상세 점검 결과, Explain Plan, Execution Plan, 인덱스 자동 설계 기능을 제공하며 개발자 스스로 튜닝이 어려운 경우 튜닝 담당자에게 튜닝을 요청할 수 있도록 지원합니다.(메뉴: 성능 점검 > SQL 셀프 성능 점검)

Q. 화면 속도가 느려졌어요. 크롬 브로우저를 종료하고 재로그인하여 사용하면 속도가 정상이 됩니다. 원인이 무엇인가요?

A. 메뉴를 클릭할 때마다 상단 탭에 메뉴가 활성화됩니다. 해당 화면을 조회했을 경우 조회된 데이터가 메모리에 저장되며 조회한 화면이 많을수록 메모리를 많이 차지하여 속도가 저하되는 현상이 발생합니다. 상단 탭에서 불필요한 메뉴를 종료하거나, 우측 상단의 ‘Reset Workpalce‘ 버튼을 클릭하면 최초 로그인 상태의 화면으로 초기화 됩니다.
