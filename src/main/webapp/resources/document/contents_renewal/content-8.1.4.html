8. 품질 점검

8.1. SQL 표준 점검

8.1.4. SQL 표준 준수 현황

개요

개발자가 작성한 SQL표준 점검 결과에 대한 현황 및 추이를 그래프로 보여준다.

화면 구성

화면은 다음과 같은 탭으로 구성되어 있다.

검색 조건
검색 조건 검색 조건 설명
프로젝트
  • 프로젝트를 선택한다.
SQL표준점검구분
  • 형상기반SQL표준일괄점검: 형상 기반 점검결과
  • 실행기반SQL표준일괄점검: 실행SQL 기반 점검결과
기준일자
  • 기준일자를 선택한다.
그리드 Area

공통적으로 각 차트의 그래프에 마우스 오버시 항목의 상세 설명이 나타난다.

표준 준수율 추이

품질검토 수행일자별 표준 준수율, 프로그램 본수, 미준수 건수 추이를 그래프로 보여준다.

컬럼 컬럼 설명
기본축
  • 프로그램본수, 미준수 건수
보조축
  • 표준 준수율 퍼센트
가로축
  • 수행일자

업무별 현황

최종 수행된 품질검토결과에 대한 프로젝트 전체의 SQL 본수, 미준수 본수, 표준 준수율, SQL 증감, 미준수 증감, 준수율 증감 현황 및 업무별 금회차/전회차 준수율, 프로그램 본수, 미준수 건수 현황을 그래프로 보여준다.

컬럼 컬럼 설명
기본축
  • SQL 본수
보조축
  • 표준 준수율 퍼센트
가로축
  • 업무명
SQL 본수
  • 총 SQL 본수
미준수 본수
  • 미준수 SQL 본수
표준 준수율
  • 표준 준수율
SQL 증감
  • 전회차 대비 SQL본수 증감을 나타낸다.
미준수 증감
  • 전회차 대비 미준수 SQL의 증감을 나타낸다.
준수율 증감
  • 전회차 대비 표준 준수율의 증감을 나타낸다.
금회차
  • 금회차 일자를 보여준다.
전회차
  • 전회차 일자를 보여준다.

품질지표별 미준수 현황

최종 수행된 품질검토 결과에 대한 프로젝트 전체 / 업무별 품질 지표별 미준수 건수를 그래프로 보여준다.

컬럼 컬럼 설명
기본축
  • 품질지표 항목
가로축
  • 품질지표 미준수 건수

업무별 표준 준수율 추이

품질검토 수행 일자별 프로젝트 전체 / 업무별 표준 준수율 추이를 그래프로 보여준다.

컬럼 컬럼 설명
기본축
  • 표준 준수율 퍼센트
가로축
  • 품질점검 수행일자

품질지표별 미준수 추이

품질검토 수행 일자별 프로젝트 전체 / 업무별 품질지표 단위의 미준수 건수 추이를 그래프로 보여준다.

컬럼 컬럼 설명
기본축
  • 표준 준수율 퍼센트
가로축
  • 품질점검 수행일자

그래프 Click

  1. 차트 그래프를 클릭한다. (마우스 커서를 해당 그래프로 옮기면 각회차 본수를 확인할 수 있다.)
  2. 선택된 업무의 현황이 하단의 그래프에 조회된다.
