7. 로그인

URL로 접속한 후에 ID/Password 를 입력하여 [Login] 버튼을 클릭한다. 아이디가 없는 경우 [신규 사용자 등록] 버튼을 클릭하여 사용자 등록을 신청한다. 신규 사용자는 관리자 승인 후에 로그인이 가능하다.

로그인 화면

신규 사용자 등록 화면

사용자 ID, 사용자명, 비밀번호, 권한은 필수 입력항목이며, 권한을 “업무개발자” 로 선택한 경우 업무 선택이 필수이다. 업무에서 업무리더는 한 명만 지정이 가능하고, 이미 업무리더가 지정되어 있으면 업무리더를 선택할 수 없고 관리자에게 업무리더 변경 요청을 해야 한다.
