14.3. 기준정보 설정

14.3.4. DB 금지 힌트 관리

모니터링 대상 DB별로 금지할 힌트의 목록을 조회하고 신규로 추가하거나 수정/삭제할 수 있다

DB금지 힌트 목록 조회

화면의 좌측에서 금지 힌트 목록을 조회할 수 있다.

DB금지 힌트 신규 추가

화면 우측에서 [초기화] 버튼을 클릭하고 신규로 추가할 금지 힌트 정보를 입력한 후 [저장] 버튼을 누른다.

DB금지 힌트 수정

화면의 좌측에 있는 목록에서 수정할 금지 힌트를 선택하면 우측에 선택한 정보가 나타나고 수정할 내용을 변경한 후 [저장] 버튼을 누른다.

DB 금지 힌트 삭제

화면의 좌측에 있는 목록에서 삭제할 금지 힌트를 선택하면 우측에 선택한 정보가 나타나고 [삭제] 버튼을 누른다.
