14.3. 기준정보 설정

14.3.7. 애플리케이션 코드 관리

애플리케이션 코드 목록을 조회하고 신규로 추가하거나 수정/삭제할 수 있다.

애플리케이션 코드 조회

화면의 상단에서 애플리케이션 코드 목록을 조회할 수 있다.

애플리케이션 코드 신규 추가

화면 하단에서 [초기화] 버튼을 클릭하고 신규로 추가할 코드 정보를 입력한 후 [저장] 버튼을 누른다.

애플리케이션 코드 수정

화면의 상단에 있는 목록에서 수정할 코드를 선택하면 하단에 선택한 정보가 나타나고 수정할 내용을 변경한 후 [저장] 버튼을 누른다.

애플리케이션 코드 삭제

화면의 상단에 있는 목록에서 삭제할 코드를 선택하면 하단에 선택한 정보가 나타나고 [삭제] 버튼을 누른다.

애플리케이션 코드 일괄 등록

다수의 애플리케이션 코드를 일괄 업로드 한다.

[애플리케이션 코드 등록] 버튼 클릭 > 템플릿 다운로드 > 템플릿 엑셀 파일에 등록할 애플리케이션 코드 입력 후 저장 > [Choose File] 버튼 클릭 > 입력한 엑셀 파일 선택 > [엑셀 업로드] 버튼을 클릭한다.

※템플릿 엑셀 파일에 애플리케이션 코드 입력 시 주의 사항

  • - 모든 항목은 필수 입력이다. 항목값 누락 시 ‘엑셀에 입력한 값들을 확인해 주세요.’라는 메시지 박스가 뜬다.
  • - 업무코드, 애플리이션 코드는 유일(PK)해야 한다. 중복 등록 시 업로드에 실패한다.
  • - 업무코드, 담당자ID는 사전에 등록되어 있어야 한다. 등록되지 않은 업무코드 또는 담당자ID로 등록 시 업로드에 실패한다.

업로드 후 성공 건수, 실패 건수, 실패 행에 대해 보여준다.
