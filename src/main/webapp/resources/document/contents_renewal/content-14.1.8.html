14. 설정

14.1. Open-POP 설정

14.1.8. 스케줄러 유형코드 관리

Open-POP 스케줄러의 유형코드 목록을 조회하고 신규로 추가하거나 수정/삭제할 수 있다

스케줄러 유형코드 목록 조회

화면의 상단에서 스케줄러 유형코드 목록을 조회할 수 있다.

스케줄러 유형코드 신규 추가

화면 하단에서 [초기화] 버튼을 클릭하고 신규로 추가할 스케줄러 유형코드 정보를 입력한 후 [저장] 버튼을 누른다.

스케줄러 유형코드 수정

화면의 상단에 있는 목록에서 수정할 유형코드를 선택하면 하단에 선택한 정보가 나타나고 수정할 내용을 변경한 후 [저장] 버튼을 누른다.

스케줄러 작업 삭제

화면의 상단에 있는 목록에서 삭제할 유형코드를 선택하면 하단에 선택한 정보가 나타나고 [삭제] 버튼을 누른다.
