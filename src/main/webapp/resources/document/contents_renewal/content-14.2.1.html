14. 설정

14.2. 사용자 관리

신규 사용자를 승인하고 기존 사용자를 변경 또는 삭제하며 사용자에게 DB와 업무 권한을 부여하고 회수하는 등의 사용자 관리업무를 처리한다.

14.2.1. 사용자 관리

신규 사용자에 대한 승인, 사용자 정보 변경, 사용자 업무 할당, 사용자 DB권한 할당 등의 관리업무를 처리한다.

사용자 목록 조회

사용자명, 사용자ID, 승인여부, 사용여부 등 검색조건을 입력하고 [조회] 버튼을 클릭한다.

기본정보관리

신규 사용자 추가, 사용자 정보 변경 등 관리업무를 수행하는 화면이다.
상단의 사용자 목록을 선택한 후에 하단에서 사용자 추가 또는 정보를 변경한다.
사용자를 삭제할 때에는 사용 여부를 [미사용] 으로 변경한 후에 저장한다.
신규 사용자 추가는 하단의 [초기화] 버튼을 클릭한 후에 입력 값을 입력하고 [저장] 버튼을 클릭한다.

사용자 비밀번호 분실 문의 시 비밀번호를 초기화 한다.
사용자를 선택하고 [비밀번호 초기화] 버튼을 클릭한다. 비밀번호 초기화 메시지 창이 뜨고 [YES] 버튼을 클릭하면 비밀번호 초기화 완료 및 해당 사용자에게 임시 비밀번호 안내에 대한 메시지 창이 뜬다.

담당자는 비밀번호 초기 후 해당 사용자에게 임시 비밀번호를 안내해 준다.

권한관리

사용자는 다중 권한을 가질 수 있으며 좌측 그리드에서 권한을 선택하여 변경하거나 [초기화] 버튼을 클릭 한 후에 새로 추가할 수 있다.

업무관리

사용자는 다중 업무를 가질 수 있으며 좌측 그리드에서 업무를 선택하여 변경하거나 [초기화] 버튼을 클릭 한 후에 새로 추가할 수 있다.

일괄 승인

승인 대상 사용자 목록을 검색하여 선택[체크]하고 [승인] 버튼을 누르면 여러 건을 일괄로 승인할 수 있다.

일괄 미승인

미승인 대상 사용자 목록을 검색하여 선택[체크]하고 [미승인] 버튼을 누르면 여러 건을 일괄로 미승인할 수 있다.

사용자 일괄 등록

템플릿을 다운로드 받아 사용자 정보를 작성하고 해당 파일을 업로드시켜 일괄로 사용자를 등록한다.
