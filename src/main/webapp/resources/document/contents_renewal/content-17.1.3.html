17.1. SQL 표준 점검

17.1.2 SQL 표준 점검 지표 조회

SQL 표준점검 지표와 품질 점검 내용 및 해결 방안에 대한 정보를 조회한다.

[Quality Inspection Index] 탭 활성화

품질점검 결과를 보기 위해서는 [Quality Inspection Index] 탭이 활성화 되어야 한다. Window > Show View > Other…를 클릭한다.

Show View 팝업창에서 “Quality Inspection Index”를 선택하고 [Open] 버튼을 클릭한다.

하단View에 [Quality Inspection Index] 탭이 활성화되면서 품질점검 지표가 조회된다.

  • Quality Inspection Index: SQL 표준 점검 지표명
  • Quality Inspection Details: SQL 표준 점검 내용
  • Solution Plan: 해결 방안
