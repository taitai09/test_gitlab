11.1.3. 튜닝 이력 조회

개요

DBIO 기반의 튜닝이력을 조회하는 화면이다. 업무 변경에 의해 과거에 튜닝한 SQL이 변경되어 재 튜닝 해야 하거나, 튜닝 가이드 반영여부를 확인할 때 유용하다.

그리드 Click

  1. 그리드 행을 클릭한다.
  2. 튜닝 이력 조회 상세 > SQL 요청 상세로 이동한다. (성능 개선 관리 SQL 요청 상세 탭 참조)
  3. SQL 개선 상세 탭을 클릭 시 SQL 개선 상세 화면이 조회된다. (성능 개선 관리 SQL 개선 상세 탭 참조)
